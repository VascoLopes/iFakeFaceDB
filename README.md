# iFakeFace

Dataset of synthetic faces modified by a GAN-fingerprint removal approach

[http:/socia-lab.di.ubi.pt/~jcneves/iFakeFace.zip](http:/socia-lab.di.ubi.pt/~jcneves/iFakeFace.zip)
